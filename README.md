# GPT-NLP-Project
# AI-Driven NLP Project using GPT-2

This project demonstrates the implementation and analysis of GPT-2, a language model, using Jupyter Notebook and Hugging Face Transformers.

## Features
- Text generation using GPT-2
- Handling incorrect grammar
- Logical reasoning analysis
- Emotional tone detection
- Research questions and conclusions

## Tools Used
- Python
- Jupyter Notebook
- Hugging Face Transformers
- GPT-2 Model

## Conclusion
GPT-2 is effective for text generation but has limitations in reasoning and factual accuracy.
